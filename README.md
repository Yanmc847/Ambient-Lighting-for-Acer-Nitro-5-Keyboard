# Ambient-Lighting-for-Acer-Nitro-5-Keyboard
Ambient Lighting for Acer Nitro 5 Keyboard  

Allows you to synchronizes your keyboard backlight with the video on your screen.  

Compatibility :
Work on Acer Nitro 5 AN515-57 and potentially with other Acer Nitro 5 laptops with 4-zone RGB keyboards  

Parameters :
All settings can be modified from the system tray icon:
FPS: How often the keyboard updates  
Maximum brightness: Maximum brightness  
Skip Threshold: Minimum color change to trigger an update  
Show RGB values: Optional display of RGB values on screen (default: OFF)  

Installation :
Go to the Releases  
Download the EXE file  
Run the EXE as Administrator  
The tool will appear in your system tray (hidden icons)
